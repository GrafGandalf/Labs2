Алгоритм 1: Средний

При нахождении нужного элемента значения после него сдвигаются вправо, а на освободившееся место ставится найденный коэффициент.
При этом изначально массив имеет N*2 количество элементов.

Алгоритм 2: Затратный

При нахождении элемента, создается массив с удвоенным найденым элементом, после чего происходит замена и удаление буфера. Каждый раз создается новый буферный массив.

Алгоритм 3: Оптимальный

Изначально элементы удваиваются (1 2 3 4) -> (1 1 2 2 3 3 4 4), после, если элемент массива не подходит под условие, то его копия удаляется.
